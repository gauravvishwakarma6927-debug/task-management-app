# Task Management App

A simple and user-friendly Task Management Application built using HTML, CSS, and JavaScript. This application helps users create, update, delete, and track tasks efficiently.

## Features

- Add new tasks
- Edit existing tasks
- Mark tasks as completed
- Delete tasks
- Responsive and clean user interface
- Real-time task tracking

## Technologies Used

- HTML5
- CSS3
- JavaScript (ES6)
